## Key Features
+ 7-day weather forecast
+ Notetaker
+ Todo list
+ High resolution background video
+ Social media and dynamic links
+ CNN news headlines

## Download
The latest version of tabyetu for Windows, Linux and macOS is downloadable from chrome Web Store
